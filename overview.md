# PR Tracker - Pull Request Dashboard

Track all your pull requests across every project in your Azure DevOps organization.

## Features

- **Organization-wide PR view** - See all your PRs across all projects in one place
- **Per-project breakdown** - Group and filter PRs by project
- **Merge conflict detection** - Instantly see which PRs have merge conflicts with detailed file-level conflict info
- **Status tracking** - Monitor approval status, reviewer votes, and comments
- **Notifications** - Get notified about merge conflicts and status changes
- **Advanced filtering** - Filter by status, reviewer, date, repository, and more
- **Beautiful UI** - Modern, clean interface with dark mode support
